The Classic game of "'Rock, Paper, Scissors'" is a foundational JavaScript Excercise. 

Given 3 variables, respectively, the user competes against the browser which randomly chooses one of the variables of its own as its choice. The fucntion compares the two inputs and determins the winner/loser of the round. 

This current version asks a prompt and logs the computer's choice and the result of the game in the console. 

Further developement could print the results to the browser, best of ___ rounds, or visuals.
